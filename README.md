# Fourier

本仓库用于集中管理曾经实现过的、分散在各处的傅里叶变换的实现，包含：

- 快速傅里叶变换（FFT）
- 离散余弦变换（DCT、DCT_2D）

未来可能加入的其他变换：

- 改进的离散余弦变换（MDCT）
- 快速数论变换（Fast NTT）
- 小波变换
- 离散正弦变换（DST）
- K-L变换，等等
