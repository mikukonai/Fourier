# Fourier

本仓库用于集中管理曾经实现过的、分散在各处的傅里叶变换的实现，包含：

- 快速傅里叶变换（FFT）及配套的复数运算
- 离散余弦变换（DCT）
- 二维离散余弦变换及配套的矩阵运算
- 改进的离散余弦变换（MDCT），用于音频编码

未来可能加入的其他变换：

- 快速数论变换（Fast NTT）
- 小波变换
- 离散正弦变换（DST）和K-L变换，用于视频编码
- 等等
